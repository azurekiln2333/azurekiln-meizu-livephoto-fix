# Flyme LivePhoto Fix（魅族实况照片修复工具）

默认文档语言：中文。
English documentation: `docs/README.en.md`

## 项目简介

这是一个基于 `PyQt6 + qfluentwidgets` 的桌面工具，用于批量处理魅族实况照片兼容性问题，支持：

- 拖拽文件/文件夹快速导入（`JPG/JPEG`）
- 批量修复勾选项
- 批量复制/移动勾选项
- 列表右键快捷操作

## 主要特性

- 仅保留 `PyQt6` 生态
- 蓝色沉浸式标题栏
- 列表支持拖拽导入（异步解析，避免主线程阻塞）
- 支持鼠标框选多行（蓝色框选）
- 支持点击表头排序
- 排序时自动清除行选中阴影（不改变复选框勾选状态）
- 输出目录默认定位到：
  - `~/Pictures/FlymeLivePhotoFix`
  - 若不存在会自动创建

## 运行环境

- Windows 10/11（推荐）
- Python 3.10+

## 安装依赖

在项目根目录执行：

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install PyQt6 PyQt6-Fluent-Widgets pywin32
```

## 启动方式

```powershell
.\.venv\Scripts\python.exe .\main_gui.py
```

## 使用说明

1. 启动程序后，确认输出目录（默认是图片目录下 `FlymeLivePhotoFix`）。
2. 将图片文件或文件夹拖拽到列表区域。
3. 勾选要处理的条目（拖拽导入默认自动勾选）。
4. 使用底部按钮执行：
   - 修复勾选项
   - 复制勾选项
   - 移动勾选项
5. 右键列表项可快速执行打开、定位、复制路径，也可进入系统右键菜单。

## 列表交互规则

- 复选框：决定是否参与修复/导出。
- 行选中阴影：仅表示当前视觉选中，不影响复选框。
- 点击表头排序后，会清除行选中阴影，但保留复选框状态。

## 常见问题

### 1) 拖拽很多文件时感觉卡顿

已使用后台线程异步解析，状态栏会显示处理进度。若目录层级过深，首轮文件遍历仍可能有短暂延迟。

### 2) 右键系统菜单弹出慢

系统原生右键菜单依赖 Shell 扩展，速度受系统环境影响。程序默认提供快速菜单，并可按需打开“系统右键菜单...”。

## 目录结构（核心）

- `main_gui_logic.py`：列表项处理、修复/导出逻辑
- `flyme_livephoto_fix_core.py`：照片识别与修复核心能力
